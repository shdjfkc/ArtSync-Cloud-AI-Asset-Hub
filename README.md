# 智绘 应用秒搭工具

## 一、项目介绍

基于 Spring Boot 3 + LangChain4j + LangGraph4j 构建应用生成平台，支持自然语言输入自动完成素材搜集、代码生成等全流程工作流。整合多级缓存、分布式限流优化性能，实现流式响应与容器化部署，生成应用可直接预览并一键部署。

<img width="1912" height="994" alt="image-20260111155305652" src="https://github.com/user-attachments/assets/4c68f956-d99a-4151-ba11-c9af37869d69" />
### 4 大核心能力

1）智能代码生成：用户输入需求描述，AI 自动分析并选择合适的生成策略，通过工具调用生成代码文件，采用流式输出让用户实时看到 AI 的执行过程。

<img width="2549" height="1313" alt="image-20260111155945287" src="https://github.com/user-attachments/assets/7bad2281-2144-4d11-8661-cd75947affcf" />

2）可视化编辑：生成的应用将实时展示，可以进入编辑模式，自由选择网页元素并且和 AI 对话来快速修改页面，直到满意为止。
<img width="2549" height="1313" alt="image-20260111155742837" src="https://github.com/user-attachments/assets/07b3fc9e-b3f9-4d2a-b543-c0b31537539a" />

3）一键部署分享：可以将生成的应用一键部署到云端并自动截取封面图，获得可访问的地址进行分享，同时支持完整项目源码下载。
