
# 绘联云 智能素材库项目介绍



## 项目介绍

基于 Spring Boot + Redis + COS 构建，支持公共 / 私有 / 团队三类图库，实现图片上传、检索、多人协作编辑。整合多级缓存、分表存储优化性能，集成 AI 扩图、以图搜图功能，已部署上线。

1）所有用户都可以在平台公开上传和检索图片素材，快速找到需要的图片。可用作表情包网站、设计素材网站、壁纸网站等：
<img width="2549" height="1313" alt="image" src="https://github.com/user-attachments/assets/9144a52f-9976-4f6a-bdbe-495e7105beaf" />

2）管理员可以上传、审核和管理图片，并对系统内的图片进行分析：
<img width="2549" height="1313" alt="image" src="https://github.com/user-attachments/assets/504dfe9f-f021-4494-b257-4ef4ed179748" />


3）对于个人用户，可将图片上传至私有空间进行批量管理、检索、编辑和分析，用作个人网盘、个人相册、作品集等：

<img width="2549" height="1314" alt="image" src="https://github.com/user-attachments/assets/2107d418-1d22-4dba-a25c-ed5cb1f74627" />

4）对于企业，可开通团队空间并邀请成员，共享图片并 **实时协同编辑图片**，提高团队协作效率。可用于提供商业服务，如企业活动相册、企业内部素材库等：
